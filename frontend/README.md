# Lead Capture PWA Frontend

## Setup

1. Instalează dependențele:
   ```bash
   npm install
   ```
2. Rulează aplicația local:
   ```bash
   npm run dev
   ```
3. Pentru build de producție:
   ```bash
   npm run build
   ```
4. Uploadează conținutul din `dist/` pe server (ex: `public_html`).

## Notă
- Pentru funcționalitate completă, backend-ul PHP trebuie să fie configurat și să aibă CORS activat dacă frontend-ul e pe alt domeniu.
- Icon-urile sunt placeholder, pot fi înlocuite cu imagini proprii.
